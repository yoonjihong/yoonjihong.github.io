## 설치하기

원하는 프로젝트명을 AppName 자리에 입력해주면된다.

```
$ npx create-react-app AppName
```

> 참고 - https://ko.reactjs.org/docs/create-a-new-react-app.html

## Typescript로 시작하기

```
$ npx create-react-app AppName --template typescript
or
$ npx create-react-app typescript-react-app --scripts-version=react-scripts-ts
```

## 실행 포트 변경하기

기본적으로 3000번 포트를 사용해서 구동되고 package.json을 수정하거나 .env 파일을 추가해서 변경할 수 있다.

<br />

##### package.json <span>실행 명령어를 아래와 같이 수정하면된다.</span>

```json
"scripts": {
  "start": "npm start -p 3001"
}
```

<br />

##### .evn파일로 변경하기 <span>.env 파일을 만들어 아래 내용을 추가하면된다.</span>

```
PORT=3001
```
