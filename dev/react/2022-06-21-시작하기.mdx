## 설치하기

원하는 프로젝트명을 AppName 자리에 입력해주면된다.

```
npx create-react-app AppName
```

> 참고 - https://ko.reactjs.org/docs/create-a-new-react-app.html

## NextJs

NextJs에서 제공하는 기능을 통해 더 큰 규모의 리액트 앱을 쉽게 생산할 수 있다.

SEO 최적화, 직관적인 파일 기반 라우팅 지원, Code Splitting등의 장점이 있다.

<br />

> NextJs는 React 프레임워크다.
